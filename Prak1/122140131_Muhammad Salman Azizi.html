Deskripsi Tugas

Tugas ini adalah membuat halaman web sederhana yang terdiri dari form input dan tabel. Berikut adalah rincian tugasnya:

  • Membuat form dengan 5 inputan teks dan satu tombol submit untuk mengirim data.
  • Membuat tabel dengan 4 kolom untuk menampilkan data secara terstruktur.
  • Form dan tabel ini dibuat tanpa menggunakan CSS eksternal, dengan fokus pada struktur HTML sederhana.

Halaman web ini merupakan bagian dari tugas pengembangan web sederhana yang menampilkan kemampuan dasar HTML dalam membuat form dan tabel tanpa styling tambahan.
